# 💳 Credit Card Transaction and Customer Dashboard using Power BI

An end-to-end Power BI project that analyzes credit card customer demographics and transaction data through interactive dashboards. The project demonstrates the complete Business Intelligence workflow—from raw data preprocessing to dashboard development.

---

## 📌 Project Workflow

This project was completed following the steps below:

### 1. Data Collection
- Started with raw **CSV datasets** containing customer and transaction information.

### 2. Data Migration
- Imported the CSV files into **SQL**.
- Used SQL to organize and manage the data for analysis.

### 3. Data Cleaning
- Removed duplicate records.
- Handled missing and inconsistent values.
- Corrected data types.
- Standardized column names and formatted the data.

### 4. Data Modeling
- Established relationships between customer and transaction tables.
- Built a clean data model for efficient analysis in Power BI.

### 5. DAX Measures
Created multiple DAX measures to calculate business KPIs, including:
- Total Revenue
- Total Interest
- Total Income
- Transaction Amount
- Transaction Count
- Customer Acquisition Cost
- Revenue by Customer Segments
- Revenue by Education
- Revenue by Job
- Quarterly Revenue
- Additional calculated measures for dashboard insights

### 6. Dashboard Development
Developed two fully interactive dashboards:

#### 📊 Credit Card Customer Dashboard
Provides insights into:
- Customer demographics
- Revenue by gender
- Age group analysis
- Salary groups
- Education levels
- Customer occupations
- Marital status
- State-wise revenue
- KPI summary cards

#### 💳 Credit Card Transaction Dashboard
Provides insights into:
- Total Revenue
- Total Interest
- Transaction Amount
- Transaction Count
- Revenue by Card Category
- Quarterly Revenue Trends
- Revenue by Chip Usage
- Customer Acquisition Cost
- Revenue by Education
- Revenue by Customer Job
- Revenue by Expenditure Type
